
# realtimerail.nyc web app


This is simple-minded Javascript web app that 
displays NYC subway times using 
[Transiter](https://www.github.com/jamespfennell/transiter).
It's written in vanilla Javascript with some assistance from the library jQuery.

The general idea is for new features in the app to be developed alongside
new features in Transiter, thus providing some direction for the infinite
possibilities that exist for Transiter.
However if the app is to get much more complicated
it would probably be worth writing it instead with the help of
React or something fancy like that.
